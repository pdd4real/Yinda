# 系统架构

音达康护系统拆分为五层。

## 物理层

家庭侧部署带麦克风的高清摄像头，支持 H.265、Wi-Fi、MicroSD、本地缓存和远程流传输。设备无需老人佩戴，适合长期独居场景。

## 数据层

主要数据表：

- `households`：老人住户、联系人、社区网格信息。
- `devices`：摄像头、拾音器、安装位置、在线状态。
- `audio_segments`：采集片段、增强前后文件、信噪比。
- `events`：事件类型、置信度、证据窗口、风险等级。
- `dispatch_tasks`：上门任务、接收人、处理结果。

## 支撑层

- 海康威视 SDK：接入音视频流和对讲。
- Stream Buffer：短窗口缓存，保证异常事件可回放。
- ATST-Frame SED：预训练自监督声音事件检测。
- TRUNet Enhancer：抑制电视声、风噪、厨房背景噪。
- YOLOv5 Fall Detector：识别跌倒、长时间趴地等姿态。

## 业务层

业务由四个模块组成：数据采集、数据处理、数据展示、系统管理。后台根据声音事件、跌倒检测、住户安全模式和联系人策略生成分级告警。

## 应用层

- 总控端：社区管理员查看地图态势、实时告警、音视频证据、派单进度。
- 小程序：家属接收告警，志愿者确认任务并反馈处理结果。
- 老人侧：无需操作，必要时通过摄像头对讲获得远程确认。
